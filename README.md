# Hraje dnes Sparta na Letné?

Jednoduchá webová aplikace, která vám řekne, zda AC Sparta Praha hraje dnes svůj domácí zápas na stadionu Letná.

## Funkce

- Zobrazí, zda Sparta dnes hraje na Letné
- Ukáže detaily zápasu (čas, soupeř, soutěž)
- Reaguje na venkovní zápasy nebo zápasy na jiných stadionech
- Elegantní design v barvách Sparty
- Funguje okamžitě bez nutnosti registrace nebo API klíčů

## Jak to rozběhnout

### Jednoduché spuštění

1. Otevřete soubor `index.html` ve webovém prohlížeči
2. Aplikace automaticky zobrazí informace o dnešním zápase

To je vše! Žádná registrace, žádné API klíče.

### Použití

Aplikace automaticky:
- Zkontroluje, zda Sparta hraje dnes
- Zobrazí detaily zápasu, pokud nějaký existuje
- Informuje vás, zda se hraje na Letné nebo jinde

## Technické informace

- Žádné závislosti, čistý HTML/CSS/JavaScript
- Používá SofaScore API pro aktuální data zápasů
- Responzivní design pro mobilní zařízení
- Bez nutnosti API klíče

## Soubory

- `index.html` - Hlavní HTML struktura
- `style.css` - Styly v barvách AC Sparta Praha
- `script.js` - Logika pro načítání a zobrazování dat

## Poznámky

- Aplikace vyžaduje připojení k internetu
- Data jsou získávána z veřejného SofaScore API
- Aplikace respektuje fair use policy (nepřetěžuje server)

## Zdroj dat

Tato aplikace používá data z [SofaScore.com](https://www.sofascore.com/)
